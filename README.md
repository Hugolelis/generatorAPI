# 🚀 Generator-API
[![License](https://img.shields.io/badge/License-MIT-yellow?style=flat-square)](LICENSE)
[![Status](https://img.shields.io/badge/Status-complete-green?style=flat-square)]()
[![Version](https://img.shields.io/badge/version-1.0.0-blue?style=flat-square)]()

A robust and scalable API built with Node.js and TypeScript, focused on delivering high-quality data generation services. This project follows industry 
best practices and clean architecture to provide reliable tools for generating various data types, such as dates, CPF , URL shortening, and more.

## ✨ Features

Data that can be generated by the **GeneratorAPI**.

* **CPF Generation.**
* **URL Shortener.**
* **Password Generator.**
* **Date Services.**
* **UUID Generation.**
* **Sorted Numbers.**
  
---

## 🛠️ Tech Stack
The main tools used in this project are:

* **Runtime:** [Node.js](https://nodejs.org/)
* **Language:** [TypeScript](https://www.typescript.org/) (Static typing for enhanced security)
* **ORM:** [Prisma](https://www.prisma.io/) (Data modeling and database queries)
* **Package Manager:** `npm`
* **Configuration:** `dotenv` for environment variables and `tsconfig` for compilation settings.

---

## 📂 Project Structure
The project organization is designed for a clear separation of concerns:

```text
├── logs/                 # Application execution logs
├── node_modules/         # Project dependencies
├── prisma/               # Prisma ORM schemas and configurations
├── src/                  # Main source code
│   ├── controllers/      # Route control logic (Request/Response)
│   ├── core/             # Core business logic
│   │   ├── generators/   # Data generation engines
│   │   └── validators/   # Business rule validators
│   ├── helpers/          # Utility functions and support
│   │   ├── errors/       # Custom error handling
│   │   ├── interfaces/   # Contract definitions (Interfaces)
│   │   ├── types/        # TypeScript specific type definitions
│   │   └── utils/        # General utility functions
│   ├── middlewares/      # Request interceptors
│   ├── repository/       # Data persistence layer
│   ├── routes/           # API endpoint definitions
│   ├── conf.ts           # Configuration file
│   └── server.ts         # Server entry point (Bootstrap)
├── .env                  # Environment variables
├── .gitignore            # Files ignored by Git
├── package-lock.json     # Dependency version lock
├── package.json          # Project manifest and scripts
├── prisma.config.ts      # Prisma specific configurations
└── tsconfig.json         # TypeScript compiler configurations
```

## ⚙️ Running

#### 1. Clone the Repository

```bash
git clone https://github.com/Hugolelis/GeneratorAPI.git
cd GeneratorAPI
```

#### 2. Install Dependencies

```bash
npm install
```

#### 3. Configure Environment Variables (.env) (example)

```bash
PORT=3000
HOST=0.0.0.0

DATABASE_URL=""
```

#### 4. Start the Server

```bash
npm run dev
// or
npm start
```
