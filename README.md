## âš™ï¸ Running

#### 1. Clone the Repository

```bash
git clone https://github.com/Hugolelis/GeneratorAPI.git
cd GeneratorAPI
```

#### 2. Install Dependencies

```bash
npm install
```

#### 3. Configure Environment Variables (.env) (example)

```bash
PORT=3000
HOST=0.0.0.0

DATABASE_URL=""
```

#### 4. Start the Server

```bash
npm run dev
// or
npm start
```
